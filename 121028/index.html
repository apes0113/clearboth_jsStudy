GIT이란?

    GIT이란 "분산형버젼관리시스템"이다.

    SVN과는 다르게 중앙서버가 아닌 로컬에서 관리한다.

    SNS서비스인 Git-hub와 함께하면 시너지 효과는 한 3985배쯤 된다.

    GIT 패키지 설치는 각자.

    GIT 저장소 추가하기

    1. 1. mkdir my_list
    2. 2. cd my_list
    3. 3. git init

    어때요? 참 쉽죠? 생각보다 git 저장소 추가는 쉽습니다.

    GIT 유저설정

    누가 커밋했는지, 계정정도는 알고 있어야죠?

    1. 1. git config --global user.name <"my_name">
    2. 2. git config --global user.email <"my_email@email.com">
    3. 3. git config --global color.ui auto

    <" ">사이에는 자신의 고유식별정보를 넣으면 된다.

    GIT의 여러 명령어들

    저장소(branch)관련 명령어

    git branch

    현재까지 생성한 branch와 내 현재 branch가 어디인지를 알려준다.

    git branch add branch_name

    branch를 생성한다.

    git branch -d branch_name

    branch를 삭제한다.

    파일 추가하기

    git add file_name OR folder_name

    git의 INDEX에 파일을 임의로 추가해둔다. (아직 커밋된게 아님)

    git commit -a -m "message blar blar blar"

    git의 HEAD에 파일을 추가한다.

    체크아웃 받기

    git checkout -b branch_name

    branch의 생성과 동시에 체크아웃을 받습니다.

    git checkout master

    다시 돌아가 볼까요?

    git push origin <branch name>

    저장소를 원격저장소로 보냅니다.
